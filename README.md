# 😊 Smiley App

Самое простое Android приложение с улыбающимся смайликом.

## Требования
- JDK 11+
- Android SDK

## Сборка
```bash
./gradlew build
```

## Запуск
```bash
./gradlew installDebug
```

## GitHub Actions
Приложение автоматически собирается при push благодаря `.github/workflows/build.yml`

Наслаждайся смайликом! 😊
